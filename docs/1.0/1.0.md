# 1.0

Version 1.0 will be a rewrite of the codebase to be more maintainable and extensible, and to support more features. The goal is to have a solid foundation for future development, and to make it easier for contributors to get involved.


Concepts:
- [Factory](#): Everything to do with looking up files, parsing them, and writing them. All the different auto-indexers and auto-menus should be sharing code, for example.
- [Regeneration](./REGENERATION.md): Comprehensive refactor to smoothly handle many asynchronous competing updates with many subscribers in a large docroot.


## Factory

The Factory is a Kingdom of Nouns concept. We'll define various agents that do work, with components acting as orchestrators, functions, and record-keepers.

Very quick, unsatisfactory description:

Some components are big, like the Dependency Grapher, which generates a dependency graph for the entire docroot. Some are medium, like the File Renderer, which is responsible for rendering a single file--including all of its many complex depeendencies. Some are small, like the Markdown Renderer, which just turns MD into HTML. The larger components tend to contain the smaller ones.
